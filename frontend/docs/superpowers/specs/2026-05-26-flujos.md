### Estados y Errores

| Punto | Error | Manejo |
|-------|-------|--------|
| Rate limit excedido | 429 Too Many Requests | Toast: "Demasiados intentos. Espera 60s." |
| Email ya registrado | Supabase error | Toast: "Este email ya está registrado." |
| Email no confirmado | Sesión sin email_verified | Puede hacer login pero se muestra banner |
| Contraseña débil | Supabase error | Toast: "La contraseña debe tener al menos 6 caracteres." |

### Estados y Errores

| Punto | Error | Manejo |
|-------|-------|--------|
| Archivo >10MB | 413 Payload Too Large | Toast: "El archivo supera 10 MB." |
| Formato no soportado | Zod validation | Toast: "Formato no soportado. Usa PDF, Excel o CSV." |
| Sin créditos | NoCreditsError | Toast: "Sin créditos. Compra un plan." + link /precios |
| Rate limit | 429 | Toast: "Demasiados análisis. Espera 60s." |
| Error de parseo | Parse error | Status = 'error', Toast: "No se pudo leer el archivo." |
| Error IA (timeout) | AbortError (30s) | Resultados sin IA (solo reglas + etiquetados) |
| Error IA (otro) | Groq API error | Fallback a solo reglas determinísticas |
| Plan Platino | - | Status = 'awaiting_payment', paywall en UI |
